# [スプーン]Spoon

[**ダウンロード**](https://github.com/eyeq/mod-1.11.2-Spoon/releases/download/1.0/1.11.2-Spoon-1.0.jar)

木の剣の上位互換であるスプーンを追加します。  
攻撃するとき、目（eye）や玉（pearl）をえぐり取ります。  
また、スイカブロックを右クリックで直接食べることができます。
